Recursos modernos no HTML 5

Versão mais atual do HTML

Novidades
    - Novas tags semânticas, como: <header>, <nav>, <section>, <article>, <aside>, <footer>, etc.
        - Tornam a página mais clara e significativa para ferramentas de acessibilidade e para mecanismo de busca (SEO)

    - Melhoreias nos elementos de formulário, como input com novos tipos (por exemplo, email, tal, date) e atributos como (require, placeholder, pattern)

    - Novas tags <audio> e <video> para incorporar áudio e video na página
