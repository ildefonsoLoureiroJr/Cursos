O que são formulários
Os formulários em HTML, são estruturas que permitem a coleta de informações dos usuários, como nome, e-mail, senha, comentários, etc.
Eles são compostos por elementos HTML que possibilitam a criação de campos de entrada, botões de envio e outras funcionalidades interativas.
Olhando de forma simples, a comunicação na web ocorre de duas formas: obtendo dados (como uma página ou uma imagem) e enviando dados. Os formulários são os principais responsáveis pela segunda.

Os formulários são compostos por: 
    - Uma tag <form> com atributos "action" e "method"
    - Campos a serem preenchidos, como <input> ou <select>
    - Um botão para enviar, ou "submeter", o formulario: <button type="submit">Texto</button>
